# 📝 NotesApp

> A fast, minimal notes application built with React + Vite.
> Deployed on AWS Amplify.

## ✨ Features

- ⚡ Lightning fast with Vite HMR
- 📝 Create, edit and delete notes
- ☁️ Deployed on AWS Amplify
- 🎨 Clean, minimal UI

```bash
git clone https://github.com/SKT100/notesapp
cd notesapp
npm install
npm run dev
```
